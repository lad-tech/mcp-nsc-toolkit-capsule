# MCP Documentation Server for nsc-toolkit

Это MCP сервер, который предоставляет документацию по фреймворку nsc-toolkit и артефакту archland для использования в контексте агентов и других систем, работающих с MCP (Model Context Protocol).

## Описание

Сервер предоставляет два инструмента:
1. `get-framework-documentation` - получает документацию по фреймворку nsc-toolkit
2. `get-archland-documentation` - получает документацию по созданию артефакта archland

## Особенности

- Предоставляет документацию в форматах markdown и JSON
- Использует официальный MCP SDK для корректной работы с протоколом
- Использует современные настройки TypeScript (esnext, nodenext)
- Использует современные типы Node.js 24
- Использует существующую документацию из проекта nsc-toolkit
- Поддерживает расширение для добавления новых инструментов

## Установка

```bash
cd mcp-server
npm install
```

## Запуск

```bash
# Для разработки
npm run dev

# Для сборки
npm run build

# Для запуска собранного кода
npm start
```

## Использование

Сервер предоставляет два инструмента для получения документации:

- `get-framework-documentation` - получает документацию по фреймворку nsc-toolkit
- `get-archland-documentation` - получает документацию по archland

Оба инструмента поддерживают параметр `format`:
- `markdown` (по умолчанию) - возвращает документацию в формате markdown
- `json` - возвращает документацию в формате JSON с заголовком и содержимым
